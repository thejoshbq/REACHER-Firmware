Detailed Description

Base class for hardware devices connected to Arduino pins.

Provides a foundation for managing a device’s pin and armed state, with virtual methods for arming and disarming that can be overridden by subclasses.

Provides a framework for managing a device’s pin and armed state, with virtual methods for arming and disarming that can be overridden by subclasses.
